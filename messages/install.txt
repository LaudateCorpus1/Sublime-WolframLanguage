
Official Sublime Text package for Wolfram Language

Thanks for installing!

Lots of features:
* Extensive syntax highlighting
* Goto Definition

And if you have the LSP package (https://github.com/sublimelsp/LSP) installed:
* Diagnostics
* Suggestions for fixes
* Formatting files / selections
* Documentation on hover
